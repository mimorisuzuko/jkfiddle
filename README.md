# JKFiddle

Test your JavaScript, SCSS and Pug on Electron. The application is inspired by [JSFiddle](https://jsfiddle.net/).

![](ss.png)

## Requirements

* Electron v1.4.6
* yarn v0.17.9

## Usage

```bash
git clone https://github.com/mimorisuzuko/jkfiddle.git
cd jkfiddle
yarn start
```
